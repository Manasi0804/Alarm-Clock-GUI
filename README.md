# Alarm-Clock-GUI
Simple Python Alarm Clock with Tkinter
This is a simple alarm clock using Python's Tkinter library.

## Features
- Set time for alarm
- Plays a sound (`sound.wav`) when the time is reached
- Uses threading for smooth GUI performance
